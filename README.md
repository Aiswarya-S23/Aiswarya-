# Aiswarya — Fashion Designer & Creative Director

A personal portfolio website for Aiswarya, built with modern web technologies and designed for dynamic content management.

## Overview

This is a sophisticated fashion designer portfolio featuring:
- Modern, responsive design
- Single Page Application (SPA) architecture
- Dynamic project showcase
- Database-driven content (in development)
- Smooth animations and transitions

## Site Pages

- **Home** – `index.html` (hero, intro, highlights)
- **Work** – Project showcase
- **About** – Designer bio
- **Contact** – Contact form
